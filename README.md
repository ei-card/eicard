# 英カード (Ei-Card) Project
---

## 概要 / Overview

**英カード (Ei-Card)** is a lightweight, open-source web application designed to help Japanese businesses, hospitality staff, and administrative offices communicate more effectively with international guests.

「伝わる、広がる、街の英語。」をコンセプトに、現場で本当に必要な接客英語・案内表示を即座に作成・表示できるツールです。翻訳アプリを起動する手間を省き、カードを選ぶだけで正確な意思疎通をサポートします。

[**View Live App**](https://ei-card.github.io/eicard/)

---

## 主な機能 / Key Features

- **Smart Search (スマート検索)** ひらがな、カタカナ、漢字、英語での曖昧検索に対応。  
  *Search by Hiragana, Katakana, Kanji, or English keywords.*

- **Fullscreen Mode (全画面モード)** スマホを掲示板代わりに。クリック一つで大きく表示。  
  *Turn your smartphone into a display board with one click for instant communication.*

- **PDF & Print (まとめて印刷)** 複数の表現を選んで、A4サイズの掲示物を一括作成。  
  *Select multiple cards and generate an A4-sized "Pack" as a lightweight, print-ready PDF.*

- **PWA (Offline Support)** オフライン対応。地下やWi-Fiのない場所でも動作。  
  *Works offline. Perfect for basements or rural areas with spotty connection.*

- **Privacy First** 全ての処理をブラウザ内で行うため、個人情報の送信は一切ありません。  
  *No data is sent to a server; everything is handled locally in the browser.*
